"# masopust" 

Nejlépe rozjet na mac os, nebo linuxu.

Jak rozjet gulp:
```
npm install -g bower
npm install -g gulp
```


Nainstalovat ruby a compass:
```
brew install ruby
gem install compass
```

Přidat compass do PATH:
```
export PATH=$PATH:/usr/local/lib/ruby/gems/2.7.0/bin/
```

Nainstalovat všechny lokální balíčky:
```
npm install
```

Nabuildovat projekt:
```
gulp
```

Spustit lokální server:
```
gulp server
```
